An interactive web-based memory game where users match countries with their corresponding flags. Designed to be both educational and engaging, with multiple regions and difficulty levels.

---

##  Demo

> Open the HTML file in your browser to play locally.

---

##  Features

-  Multiple regions:
  - Africa  
  - Asia  
  - Europe  
  - Americas  
  - Oceania  

- Difficulty levels:
  - 4x2 (Easy)
  - 4x4 (Hard)

-  Memory-based gameplay  
-  Live match counter  
-  Win detection with alert  
-  “Show Answer” feature:
  - Displays flags
  - Shows country names
  - Includes capital cities
